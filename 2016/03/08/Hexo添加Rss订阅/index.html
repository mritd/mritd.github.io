Hexo 添加 Rss 订阅

Posted by 漠然 on March 8, 2016

一、安装RSS插件

执行以下命令安装 RSS 插件

npm install hexo-generator-feed --save

二、开启网站 RSS 支持

编辑网站根目录下的 _config.yml，添加以下代码开启

# RSS订阅支持
plugin:
- hexo-generator-feed

# Feed Atom
feed:
type: atom
path: atom.xml
limit: 20

三、主题开启 RSS 支持

主题不同开启方法不同，我的是 NexT 主题，默认就可以；其他主题请参考主题说明。

四、生成 RSS

执行 hexo clean && hexo g重新生成博客文件完成部署即可。 转载请注明出处，本文采用 CC4.0 协议授权
